Vos droits

Vos choix en matière de confidentialité.

Dernière mise à jour : 24 mai 2026

Chez Thirsty Cellar, nous croyons que vos données vous appartiennent. Cette page rassemble toutes les actions que vous pouvez exercer sur vos informations personnelles, en quelques clics ou par un simple courriel.

Pour comprendre quelles données nous collectons et pourquoi, consultez notre Politique de confidentialité complète.

1. Accéder à vos données

Vous pouvez à tout moment demander une copie de toutes les données personnelles que nous détenons à votre sujet.

Comment

  • Envoyez un courriel à dev@thirstyc.com
  • Objet : « Demande d'accès à mes données »
  • Délai de réponse : 30 jours maximum
  • Format de livraison : fichier JSON structuré (lisible et portable)

2. Corriger vos données

Si certaines informations à votre sujet sont inexactes (nom mal orthographié, courriel obsolète, etc.), vous pouvez les corriger.

Comment

  • Directement dans l'application Thirsty Cellar : Profil → Modifier
  • Ou par courriel à dev@thirstyc.com avec la correction demandée

3. Supprimer votre compte et vos données

Vous pouvez demander la suppression complète de votre compte et de toutes vos données associées. Cette action est irréversible.

Comment

  • Dans l'application : Profil → Paramètres → Supprimer mon compte
  • Ou par courriel à dev@thirstyc.com avec l'objet « Suppression de mon compte »
  • Délai d'exécution : 30 jours maximum
  • Données conservées après suppression : aucune, sauf obligations légales (ex. registres de facturation Apple, conservés conformément aux lois fiscales)

4. Désactiver l'analyse d'usage (PostHog)

Nous utilisons PostHog (basé en Allemagne, conforme RGPD) pour comprendre comment l'application est utilisée et l'améliorer. Vous pouvez désactiver cette collecte à tout moment.

Comment

  • Dans l'application : Profil → Paramètres → Confidentialité → Désactiver l'analyse d'usage
  • Le rapport de crashs (Sentry) continue de fonctionner — il est essentiel pour corriger les bugs et n'identifie pas individuellement les utilisateurs

5. Refuser l'analyse d'étiquettes par IA

La fonctionnalité de scan d'étiquettes envoie temporairement la photo à un service d'IA (Qwen, basé en Chine) pour identification. La photo n'est pas conservée.

Comment

  • Vous pouvez choisir de ne jamais utiliser cette fonctionnalité (saisie manuelle des informations de vin)
  • Avant chaque scan, l'application demande votre consentement explicite
  • Vous pouvez révoquer ce consentement à tout moment dans : Profil → Paramètres → Confidentialité

6. Télécharger vos données (portabilité)

Vous avez le droit de recevoir vos données dans un format structuré et portable, pour les transférer vers un autre service si vous le souhaitez.

Comment

  • Envoyez un courriel à dev@thirstyc.com
  • Objet : « Demande de portabilité de mes données »
  • Format de livraison : fichier JSON structuré
  • Délai : 30 jours maximum

7. Refuser la vente de vos données (CCPA/CPRA, Californie)

Thirsty Cellar ne vend aucune de vos données personnelles à des tiers, peu importe le contexte. Cette pratique est interdite chez nous, indépendamment des exigences californiennes.

Si vous résidez en Californie et souhaitez confirmation formelle de ce statut, écrivez à dev@thirstyc.com avec l'objet « Confirmation Do Not Sell ».

8. Retirer votre consentement

Pour toutes les données traitées sur la base de votre consentement (analytics, scan d'étiquettes IA, etc.), vous pouvez retirer ce consentement à tout moment.

Comment

  • Pour les analytics : voir Section 4
  • Pour le scan d'étiquettes : voir Section 5
  • Pour tout autre consentement : courriel à dev@thirstyc.com

Le retrait du consentement n'affecte pas la légalité des traitements effectués avant ce retrait.

9. Déposer une réclamation

Si vous estimez que vos droits ne sont pas respectés, vous pouvez déposer une plainte auprès de l'autorité compétente :

  • Québec : Commission d'accès à l'information du Québec (CAI) — cai.gouv.qc.ca
  • France : Commission nationale de l'informatique et des libertés (CNIL) — cnil.fr
  • Union européenne : autorité de protection des données de votre pays
  • Californie : California Privacy Protection Agency (CPPA) — cppa.ca.gov
